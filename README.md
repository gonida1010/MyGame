Gontress 🚀
2D 턴제 포트리스 스타일의 대전 게임 (Pygame으로 제작)

(👆 게임 플레이 GIF나 스크린샷을 이곳에 추가하세요!)

🎮 게임 소개
Gontress는 Pygame으로 제작된 2D 턴제 포트리스 스타일의 대전 게임입니다. 웜즈(Worms)와 유사하게, 플레이어는 포물선으로 발사체를 쏘아 지형을 파괴하고, 강력한 넉백으로 상대를 맵 밖으로 떨어뜨려 승리해야 합니다.

이 프로젝트는 Pygame의 기본 기능과 물리 엔진, 스프라이트 관리, 게임 상태 관리 등을 학습하고 구현하는 것을 목표로 합니다.

✨ 주요 기능
3종류의 개성 있는 캐릭터: 레드, 블루, 그린 중 하나를 선택하여 플레이할 수 있습니다.

고유 스킬 시스템: 2단계 게이지(보너스 샷)를 성공시키면 캐릭터별 고유 스킬이 발동합니다.

레드 (김사과): 3발 연속 발사로 한 지점을 집중 공격합니다.

블루 (반하나): 강력한 광역 폭발(대형 폭탄)로 넓은 지형을 파괴합니다.

그린 (이메론): 발사체가 비행 중(혹은 시간 경과 후) 3갈래로 분리되어 여러 지점을 견제합니다.

다양한 맵과 테마: 게임 시작 시 평원, 구룽지(바위 지대), 설원 3가지 맵 중 하나가 랜덤으로 선택됩니다.

맵마다 다른 배경 이미지와 지형 타일 색상이 적용됩니다.

파괴 가능한 지형: 모든 지형은 발사체에 의해 파괴되며, 맵의 형태가 실시간으로 변합니다.

공중 발판: 맵이 공중에 떠 있는 형태로, 게임이 루즈해지지 않고 빠르게 진행됩니다.

게임 편의 기능:

캐릭터 선택창: 게임 시작 전 P1과 P2가 캐릭터를 선택할 수 있습니다.

재시작: 게임오버 화면에서 <kbd>R</kbd> 키를 눌러 캐릭터 선택창으로 돌아갈 수 있습니다.

디테일한 시각 효과:

캐릭터 이동 방향에 따른 좌우 이미지 반전

캐릭터별 개별 y_offset 적용으로 정교한 지형 착지

🕹️ 조작 방법 및 게임 흐름
캐릭터 선택: 게임이 시작되면 P1과 P2가 각각 캐릭터를 선택합니다.

이동 (MOVE): 5초간 자유롭게 이동합니다.

P1 (휴먼): <kbd>A</kbd> (왼쪽), <kbd>D</kbd> (오른쪽)

P2 (휴먼): <kbd>←</kbd> (왼쪽), <kbd>→</kbd> (오른쪽)

조준 (AIM 1): 이동 시간이 끝나면 자동으로 각도 조절이 시작됩니다.

P1 (휴먼): <kbd>Space</kbd> / P2 (휴먼): <kbd>Enter</kbd> 를 눌러 1단계 각도를 확정합니다.

보너스 샷 (AIM 2): 캐릭터 옆에 2단계 게이지가 나타납니다.

타이밍에 맞춰 키를 다시 누르면 스킬(보너스 샷)이 발동합니다. (실패 시 일반 발사)

승리: 상대방을 맵 아래로 떨어뜨리면 승리합니다.

재시작: 게임오버 화면에서 <kbd>R</kbd> 키를 누르면 캐릭터 선택창으로 돌아갑니다.

🛠️ 설치 및 실행 방법
이 저장소를 클론(Clone)하거나 다운로드합니다.

Bash

git clone https://github.com/your-username/gontress.git
cd gontress
필요한 라이브러리(Pygame)를 설치합니다.

Bash

pip install pygame
게임을 실행합니다. (파이썬 파일 이름이 main.py라고 가정)

Bash

python main.py
